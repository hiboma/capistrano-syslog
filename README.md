Capistrano Syslog
==========================

capistrano plugin to log revison by syslog

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'capistrano-syslog'
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install capistrano-syslog
```

Usage
-----

Capfile:

```ruby
require 'capistrano/syslog'
```
